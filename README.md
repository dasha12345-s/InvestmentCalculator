# Investment Calculator

The Investment Calculator is a web-based application that allows users to estimate their investment growth over time based on input parameters such as initial investment, annual contribution, rate of return, and investment duration. The project uses React for the frontend and Vite for build tooling.

## Features

- **Investment Estimation**: Calculate the future value of investments based on user inputs.
- **Dynamic Updates**: Automatically update the estimated future value as input values change.
- **User-Friendly Interface**: Intuitive and easy-to-use interface for entering investment parameters.

## Technologies Used

- **Frontend**: React
- **Build Tooling**: Vite
- **Languages**: JavaScript, CSS, HTML

## Usage

1. **Enter Initial Investment**: Specify the initial amount invested.
2. **Specify Annual Contribution**: Set the annual contribution amount.
3. **Set Rate of Return**: Define the expected rate of return.
4. **Define Investment Duration**: Set the investment duration in years.
5. The estimated future value of the investment is calculated and displayed automatically as values change.

## Code Overview

### /UserInput.jsx

Handles the user input for investment parameters.

### /Results.jsx

Displays the calculated results based on the user inputs.

### /App.jsx

Main application component that integrates the UserInput and Results components and manages the overall layout and state.
